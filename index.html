Política de Privacidade - QFocusTime

Última atualização: 20 de Outubro de 2025

1. Introdução

A QEQSoftLab ("nós", "nosso" ou "nos") opera o aplicativo móvel QFocusTime (o "Aplicativo"). Esta página informa sobre nossas políticas em relação à coleta, uso e divulgação de dados pessoais quando você usa nosso Aplicativo.

Ao usar o Aplicativo, você concorda com a coleta e uso de informações de acordo com esta política.

2. Informações que Coletamos

2.1 Informações Fornecidas por Você

Autenticação (Opcional)

  • Email (quando você faz login com Google ou Apple)
  • Nome (quando você faz login com Google ou Apple)
  • ID único de autenticação

Dados de Uso Local

  • Sessões Pomodoro (metas, durações, anotações)
  • Configurações do aplicativo
  • Estatísticas de produtividade
  • Progresso de gamificação (XP, conquistas, streaks)

2.2 Informações Coletadas Automaticamente

Google AdMob (Anúncios)

  • Identificador de publicidade (Advertising ID)
  • Endereço IP
  • Dados de dispositivo (modelo, sistema operacional, versão)
  • Dados de uso do app (tempo de sessão, interações com anúncios)

Google Play Billing (Compras In-App)

  • Histórico de compras
  • Status de assinatura
  • ID de transação

3. Como Usamos Seus Dados

3.1 Dados Locais

  • Sessões e Metas: Armazenados localmente no seu dispositivo para exibir seu histórico e estatísticas
  • Configurações: Salvas localmente para personalizar sua experiência
  • Gamificação: Calculado e armazenado localmente para motivar seu progresso

3.2 Autenticação

  • Email: Usado apenas para identificar sua conta e permitir login futuro
  • Não compartilhamos seu email com terceiros

3.3 Publicidade (AdMob)

O Google AdMob usa os dados coletados para:

  • Exibir anúncios personalizados relevantes
  • Medir a eficácia dos anúncios
  • Prevenir fraudes

Você pode optar por não receber anúncios personalizados:

  • Android: Configurações > Google > Anúncios > Desativar personalização de anúncios
  • Ou adquirir a versão Premium (sem anúncios)

3.4 Compras In-App

  • Processar sua assinatura Premium
  • Restaurar compras em novos dispositivos
  • Gerenciado pelo Google Play Billing

4. Compartilhamento de Dados

4.1 Com Terceiros

Google AdMob

  • Compartilhamos dados de dispositivo e uso para exibição de anúncios
  • Política de Privacidade: https://policies.google.com/privacy

Google Play Services

  • Para autenticação e compras in-app
  • Política de Privacidade: https://policies.google.com/privacy

Apple Sign-In (iOS apenas)

  • Para autenticação
  • Política de Privacidade: https://www.apple.com/legal/privacy/

4.2 Não Vendemos Dados

  • NUNCA vendemos seus dados pessoais para terceiros
  • NUNCA compartilhamos seu email com anunciantes

5. Armazenamento de Dados

5.1 Dados Locais

  • Todos os dados de sessões, metas e estatísticas são armazenados localmente no seu dispositivo
  • Não enviamos essas informações para servidores externos
  • Se você desinstalar o app, esses dados serão perdidos

5.2 Dados na Nuvem (Futuro)

  • Planejamos implementar backup em nuvem opcional
  • Será totalmente opcional e requer consentimento explícito
  • Você poderá excluir seus dados na nuvem a qualquer momento

6. Segurança

Valorizamos sua privacidade e implementamos medidas de segurança:

  • Autenticação via OAuth (Google/Apple) - não armazenamos senhas
  • Dados locais protegidos pelo sistema Android/iOS
  • Comunicações via HTTPS
  • Não coletamos dados sensíveis desnecessários

7. Seus Direitos (LGPD/GDPR)

Você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incorretos
  • Excluir seus dados
  • Revogar consentimento para coleta de dados
  • Portabilidade dos seus dados

Como Exercer Seus Direitos

  • Excluir conta: Menu > Configurações > Fazer Logout (deleta dados locais)
  • Desativar anúncios personalizados: Configurações do dispositivo
  • Solicitar exclusão de dados: Entre em contato conosco

8. Dados de Menores

O QFocusTime não coleta intencionalmente dados de menores de 13 anos. Se você é pai/mãe e descobre que seu filho forneceu dados pessoais, entre em contato conosco.

9. Alterações nesta Política

Podemos atualizar nossa Política de Privacidade periodicamente. Notificaremos você sobre mudanças significativas:

  • Publicando a nova política nesta página
  • Atualizando a data de "Última atualização"
  • Notificação no aplicativo (para mudanças importantes)

10. Exclusão de Dados

Como Excluir Seus Dados

Dados Locais (Sessões, Metas, Estatísticas)

  • Desinstale o aplicativo do seu dispositivo
  • Ou: Menu > Configurações > Limpar Dados (se implementado)

Conta de Autenticação

  1. Abra o app
  2. Menu (três pontos) > Fazer Logout
  3. Isso remove a associação da sua conta

Dados de Anúncios (AdMob)

  • Android: Configurações > Google > Anúncios > Redefinir ID de publicidade
  • Ou redefina o ID de publicidade nas configurações do Google

Solicitar Exclusão Completa
Entre em contato conosco em: suporte@qeqsoftlab.com

11. Transferência Internacional de Dados

Seus dados podem ser transferidos para servidores do Google (AdMob, Play Services) localizados fora do Brasil. Garantimos que essas transferências seguem padrões de segurança adequados (LGPD/GDPR).

12. Base Legal para Processamento (LGPD)

Processamos seus dados com base em:

  • Consentimento: Autenticação, anúncios personalizados
  • Execução de contrato: Compras in-app, fornecimento do serviço
  • Interesse legítimo: Melhoria do app, análise de uso

13. Contato

14. Resumo Simplificado

O que coletamos:

  • Email (se você fizer login)
  • Dados de uso local (sessões, metas - armazenados no seu dispositivo)
  • Dados de anúncios (via Google AdMob)

O que NÃO fazemos:

  • ❌ Não vendemos seus dados
  • ❌ Não compartilhamos seu email
  • ❌ Não acessamos seus dados locais

Como protegemos você:

  • ✅ Dados de sessões ficam apenas no seu dispositivo
  • ✅ Autenticação segura (Google/Apple)
  • ✅ Você pode desativar anúncios personalizados
  • ✅ Versão Premium sem anúncios disponível

Seus direitos:

  • Acessar, corrigir ou excluir seus dados a qualquer momento
  • Desativar coleta de dados (anúncios personalizados)
  • Solicitar exclusão completa da conta
